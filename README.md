## 利用二维数组验证均匀分布 ##


根据输入的行数和列数生成不同的界面，然后根据随机生成的是并且将每一行每一列的数随机相加求出和，当该运算进行足够长时间时，我们就会看出最后一行和最后一列的值大体上是一样的，从而验证均匀分布。
